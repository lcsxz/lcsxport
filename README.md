# LCsX Port
----

Olá seja bem vindo ao meu projeto!

O Principal objetivo deste projeto foi construir uma biografia profissional com um portfolio!

# Linguagens e Tecnologias utilizadas!
----

-HTML
-CSS
-JS
-NodeJS (Express JS)
